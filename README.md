# Northcoders News (dentednerd edition, created July 2017)

## Prerequisites

- run NCNews-BE; API will be accessible on port 8000

## Installation and Usage

1. `git clone https://github.com/dentednerd/NCNews-FE.git`
1. `cd NCNews-FE`
1. `yarn`
1. `yarn start`
